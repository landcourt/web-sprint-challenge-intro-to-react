What is React JS and what problems does it solve? Support your answer with concepts introduced in class and from your personal research on the web.

React is a javascript library for building user interfaces. The main problem solved my react is the developmentof large applications that have temporal data changes.

Describe component state.

React components have a build in state object. That state object is where you store property values that belong to the component.

Describe props.

Props is a special keyword in react, which stands for properties and is used for passing data from one component to another.

What are side effects, and how do you sync effects in a React component to changes of certain state or props?

Side effects are anything that affects something outside of the scope of the current function thats being executed. To sync effects you'd use the hook useEffect, this way you can handle lifecycle events directly inside function components.
